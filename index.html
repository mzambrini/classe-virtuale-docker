--- layout: home title: Esercitazioni con Docker - Laboratorio virtuale ---

Con Play with Docker è possibile eseguire laboratori e tutorial in un ambiente virtuale che ci mette a disposizione tutti gli strumenti forniti da Docker. Si potranno creare containers, immagini e, più in generale, sperimentare tutti i comandi docker.

Il solo browser è più che sufficiente e non c'è alcuna necessità di installare Docker sulle propria postazione.

L'unico prerequisito richiesto è avere un'utenza registrata (gratuitamente) sul sito https://hub.docker.com/

Si consiglia di eseguire le lezioni nell'ordine proposto, in quanto ogni lezione è propedeutica alla successiva

Lista dei tutorial

  • Lezione 1: Ciao mondo (e dintorni)

    • Introduzione ai concetti base di Docker. Questa lezione include:

    • Fondamenti di Docker
    • Utilizzo del comando docker run (Ciao mondo)
    • Come leggere i logs di un container
    • Come individuare i container in esecuzione e/o stoppati
    • Come connettersi a un container in esecuzione

  • Lezione 2: Creazione e configurazione di un server nginx

    • Utilizzo di un'immagine nginx per esporre attraverso un servizio HTTP i nostri contenuti HTML. Gli argomenti inclusi sono:

    • Creazione e avvio del container nginx
    • Fermo e rimozione del container
    • Mapping randomico delle porte
    • Utilizzo dei volumi

  • Lezione 3: Fondamenti di networking

    • Introduzione ai concetti basilari del networking con Docker. Questa lezione include:

    • Il networking su Docker
    • Attestazione di containers sulla rete bridge di default
    • Utilizzo del comando ping per verificare la connettività tra containers
    • Il flag --link
    • Creazione di una rete bridged user-defined

  • Lezione 4: Creiamo le nostre immagini

    • Introduzione alla scrittura di Dockerfiles. Questa lezione include:

    • Introduzione ai Dockerfiles
    • Build e analisi di immagini
    • Inizializzazione dei containers
    • Binding di folder locali all'host
    • Binding utilizzando volumi docker

  • Lezione 5: Usiamo docker-compose per gestire gruppi di containers

    • Introduzione alla gestione di gruppi di containeres comunicanti e cooperanti tra di loro su singolo host. Questa lezione include:

    • Cenni sul tool docker-compose
    • Attestazione di containers sulla rete bridge di default
    • Utilizzo del comando ping per verificare la connettività tra containers
    • Il flag --link
    • Creazione di una rete bridged user-defined

  • Lezione 6: Docker e WebSphere

    • Creazione di un'immagine customizzata di WebSphere 9.0.0.6 con la nostra applicazione JavaEE correttamente deployata. Questa lezione include:

    • Cenni sulle immagini WebSphere messe a disposizione da IBM
    • Dockerfile con clausole FROM multiple
    • Provisioning dell'immagine WebSphere customizzata
    • Creazione e test del container

  • Lezione 7: Docker e Maven

    • Utilizzo di docker per effettuare build con Maven. Questa lezione include:

    • Approccio stateless all build con Maven di progetti java
    • Caching del repository locale
    • Considerazioni e possibili evoluzioni dell'approccio proposto
